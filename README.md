# frontend-2024

游戏客服工单系统

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

游戏客服工单系统
采用vite+antd+formily实现的，因为vite基于es modules，打包速度快，故采用。
formily是阿里基于antd开源的表单系统，适合做表单开发。
